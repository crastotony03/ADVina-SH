# ADVina-SH
Shell script for virtual screening developed for autodock vina v1.2.7

This is experimental project developed to support a specific research workflow. It may require modifications to run reliably across different systems.

To convert molecules using openbabel execute obabel.sh (Only work on linux systems. use "sudo apt install openbabel" to install)  
To dock execute config.sh (add alias to config.sh for easy access)
